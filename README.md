# 🏥 Smart Clinic Assistant

A modern AI-powered healthcare management system built using **Python, Streamlit, and SQLite**.

Smart Clinic Assistant simplifies clinic operations while helping patients manage appointments, medicines, medical reports, emergency contacts, and preventive healthcare—all from a single application.

## 🚀 Installation

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/meetgandhi241-debug/smart-clinic-assistant.git
cd smart-clinic-assistant
```

Create virtual environment

```bash
python -m venv venv
```
Activate

Windows

```bash
venv\Scripts\activate
```

macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
